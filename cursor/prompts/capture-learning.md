Capture the most important durable learning from the current task as an Obsidian-compatible Markdown note.

Requirements:

- Use YAML frontmatter.
- Use the capture template (`obsidian/templates/capture-note-template.md`).
- Emphasize what changed in understanding.
- Include only evidence that will matter later.
- Add tags and likely wikilinks.
- Default status: inbox.
- Resolve the vault and default `inbox/` path per `AGENTS.md`.
- Output only the note.
